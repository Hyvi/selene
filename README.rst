Selene
======

A simple CMS for blogging inspired by my beautiful wife and built with Tornado
and MongoDB.

Requirements
------------

* `Tornado`_
* `PyMongo`_
* `py-bcrypt`_
* `WTForms`_
* `docutils`_
* `TornadoMail`_
